# Theme 4: Space Station / Sci-Fi

**Aesthetic:** Clean, modular space station hardware. White and gunmetal panels, holographic displays, glowing conduits. NASA-meets-Star-Trek — functional, clinical, futuristic. Each component is a module you'd find on an orbital platform.

**Palette:** White/light gray hull panels, gunmetal/dark gray structural frames, orange caution accents, holographic blue readouts, cyan energy conduits for highlights (health layer).

**Shared traits:** Every component is a self-contained pod or module with standardized docking ports. Holographic status readouts float above each unit. Cyan energy conduits pulse along edges as the health layer. Clean, geometric silhouettes.

---

## Components

### Server
A compute pod — cylindrical white module with a transparent dome showing spinning holographic data cores inside. Antenna array on top. Blue status ring around the base.

### Database
A cryo-storage vault — hexagonal armored module with a heavy blast door. Frost crystals on the seams. A holographic inventory manifest floats beside the access panel. Cyan cryo-glow seeps from the edges.

### Data Cache
A quantum buffer crystal — a hovering crystalline shard in a magnetic containment ring. Smaller and faster than the vault. Cyan energy arcs between the crystal facets.

### Load Balancer
A traffic control array — a flat dish module with multiple articulating antenna arms. A holographic display shows routing paths. Arms swivel and redirect signal beams (cyan traces).

### CDN
A relay satellite — a compact module with solar panel wings and a broadcast dish. Orbits closer to the edge. Cyan signal rings pulse outward from the dish in concentric waves.

### API Gateway
An airlock module — reinforced door with a biometric scanner panel. Status lights show green (authorized) or red (denied). A force field shimmer (cyan) seals the threshold.

### Queue
A cargo conveyor — a transparent tube module with glowing data packets lined up single-file inside, waiting to be processed. A counter display shows queue depth. Cyan glow on each packet.

### Worker
A robotic assembly arm — a mechanical arm pod with articulating joints, actively welding/building. Cyan sparks at the tool tip. A task status hologram shows the current job progress.

### Streaming Server
A broadcast tower — tall antenna spire with rotating dish arrays. Continuous cyan signal waves emanate in all directions. A "LIVE" indicator pulses on the base display.

### Edge Cache
A drop pod cache — a small re-entry capsule module deployed close to the action. Compact, armored, fast-access hatch. Cyan heat-shield glow on the hull edges.

### DNS/GTM
A star chart navigator — a holographic star map table with plotted routes between systems. Cyan navigation lines connect waypoints. A central compass gyroscope spins in the display.

### Blob Storage
A cargo bay container — oversized modular shipping container with magnetic clamps. Much larger than other storage modules. Reinforced hull, industrial yellow hazard stripes, cyan scan lines on the manifest panel.

### Circuit Breaker
An emergency bulkhead — a thick blast door module with red warning lights and a manual override lever. When triggered, cyan emergency force fields seal the section. Klaxon speaker on top.
